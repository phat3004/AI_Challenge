# Challenge 01 — Insurance Plan Comparison Page

This is a side-by-side plan comparison page built for **Papaya Insurance** using **HTML5, Tailwind CSS, Outfit and Inter Google Fonts**, featuring premium glassmorphism, responsive grid, dynamic pricing toggle (monthly/annual), and value highlight badges.

## Features

- **Side-by-Side Comparison**: Clear columns for Bronze, Silver, and Gold tiers.
- **Premium Aesthetic**: Dark mode design featuring radial gradients, glowing accents, and glassmorphism styling (`backdrop-filter`).
- **Billing Cycle Toggle**: Switch between monthly billing and annual billing, applying a 10% discount on the fly.
- **Responsive Layout**: Table format on desktop, turning into easy-to-read stacked cards on mobile devices.
- **Value Highlighting**: Best in class parameters (lowest copay, highest limit, no waiting periods) are dynamically highlighted in emerald colors.
- **"Recommended" Badge**: Silver plan displays a prominent "Recommended" badge based on its superior value-for-money ratio.

## Getting Started

Since this is built using standard web technologies with client-side bundling, you can view the application instantly without any local dependencies or installation steps.

### Method 1: Double-click HTML (Direct File Open)
Simply double-click the `index.html` file or drag it into any modern web browser (Chrome, Safari, Edge, Firefox).

```bash
open index.html
```

### Method 2: Serve Locally (Optional)
If you want to serve it locally using a lightweight server, run one of the following commands in this directory:

**Using Python:**
```bash
python3 -m http.server 8000
```
Then open [http://localhost:8000](http://localhost:8000) in your browser.

**Using Node.js (npx):**
```bash
npx serve .
```
Then open the local URL printed in the terminal.
